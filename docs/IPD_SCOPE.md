\# IPD Feature Scope (Frozen)



\## Included in Interim Demo

\- Pomodoro timer with configurable study and break durations

\- Progress tracking based on completed study sessions

\- FocusNest with single prompt-response interaction



\## Excluded from Interim Demo

\- User authentication

\- Persistent user profiles

\- AI memory or conversation history

\- Notifications and analytics

\- Mobile optimisation



This scope is frozen for the Interim Progress Demonstration.
